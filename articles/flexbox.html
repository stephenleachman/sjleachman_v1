All About FlexBox

Define A Flex Container

When using flexbox the first step is to define a container (A parent element that nests child elements) These child elements are then called Flex Items. To define a flex-container, the CSS properties are:

.container {
display: flex;
}
Flex Direction

Controlling the Direction of Flex Items inside the container working with the properties of the main container we can set the flex-direction. Note that Row is the default value. There are more direction that we can set:

.container {
display: flex;
flex-direction: row;
}
.container {
display: flex;
flex-direction: row-reverse; /* To reverse the direction flex items in a row */
}
.container {
display: flex;
flex-direction: column /* rotates the main axis so that flex items are laid out vertically. */
}
.container {
display: flex;
flex-direction: column;-reverse; /* Like the row-reverse property, you can swap the top-to-bottom direction of a column with the valule */
}
Flex Wrap

The flex-wrap property allows flex containers to break items onto multiple lines and allow them to wrap as needed. Note that the flex-wrap property is for flex containers only:

.container {
display: flex;
flex-direction: row;
flex-wrap: wrap;
}
Justify Cntent

Distributing Space Inside a Flex Container can be done with the justify-content property which lets you control the position and alignment of flex Items on the main axis and how space should be distributed in a flex container. Note: The justify-content property to flex containers only.

.container {
display: flex;
flex-direction: row;
flex-wrap: wrap;
justify-content: flex-start; /* The default value which places items towards the start of each flex line */
}
.container {
display: flex;
flex-direction: row;
flex-wrap: wrap;
justify-content:flex-end; /* place items at the end of the flex container */
}
.container {
display: flex;
flex-direction: row;
flex-wrap: wrap;
justify-content:center; /* place items at the center of the flex container */
}
.container {
display: flex;
flex-direction: row;
flex-wrap: wrap;
justify-content:space-between; /* displays equal spacing between flex items */
}
.container {
display: flex;
flex-direction: row;
flex-wrap: wrap;
justify-content:space-around; /* displays equal spacing around every flex item */
}
Order

Changing the Order of Flex Items. By default, flex items are laid out in the order they appear in the source code. We can use the order property to change the order of any flex item. Note: The order property applies to the flex item only.

.item 1 {
order: 0; /* Expands flex items to take up the full space of a line. */
}
.item 2 {
order: -1; /* The higher the flex-grow value, the more an item grows relative to the other items. */
}
Flex Grow

Growing Flex Items. Flex items can grow or shrink in relation to other flex items and the available space inside the flex container. Note the flex-grow value is by default set to 0 and can only be applied to the flex-items

.items {
flex-grow: 1; /* Expands flex items to take up the full space of a line. */
}
.item-1 {
flex-grow: 3; /* The higher the flex-grow value, the more an item grows relative to the other items. */
}
Flex Basis

The flex-basis property gives you even more control over the size of flex items, and the flex shorthand property intelligently sets flexbox values when you don't specify them. Note: flex and flex-basis can only be applied to flex items.

.items {
flex-grow: 1;
flex-basis: 150px; /* The flex-basis except values the same as width and high property's (px, em, %, ext.) */
}
.items {
flex: 1 150px; /* same as above. Flex = ( flex-grow, flex-basis and flex-shrink. )*/
}
Align Items

Align flex items on the cross axis with the align-items and align-self properties. Note: the align-items aligns flex items vertically in the flex container and can only be applied to the flex container, whereas the align-self is a property that can be used on individual flex items.

.container {
display: flex;
flex-direction: row;
flex-wrap: wrap;
justify-content: flex-start;
height: 400px;
align-items: flex-start;/* aligns all flex items to the start of the cross axis */
}
.container {
display: flex;
flex-direction: row;
flex-wrap: wrap;
justify-content: flex-start;
height: 400px;
align-items: flex-end; /* aligns all flex items to the end of the cross axis */
}
.container {
display: flex;
flex-direction: row;
flex-wrap: wrap;
justify-content: flex-start;
height: 400px;
align-items: center; /* aligns all flex items to the center of the cross axis */
}
.item-2 {
align-self: flex-start /* aligns a flex item to the start of the cross axis. */
}
.item-2 {
align-self: flex-end; /* aligns a flex item to the end of the cross axis. */
}
.item-2 {
align-self: center; /* aligns a flex item to the end of the cross axis. */
}
Vertical and Horizontal Centering

three ways to center-align elements using flexbox

Set the flex container's justify-content and align-items values to center:

.container {
justify-content: center;
align-items: center;
}

Set the flex item's margin value to auto:

.items {
margin: auto;
}

Set the flex container's justify-content value to center, while setting the flex item's align-self value to center:

.container {
justify-content: center;
}
.items {
align-self: center;
}
